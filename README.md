# Run Locally
1. Clone this repo
   git clone <repo_url>.git
2. `npm install`
3. Set the API KEY in .env file
4. `npm run dev`

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
